# Roadmap

## v0.1

- npm workspace CLI.
- zod config validation.
- Playwright generated specs.
- Basic visual snapshots, mutation score, offline triage, and GitHub markdown.
- Demo React app.

## v0.2

- Tolerance-based PNG visual diffing.
- Rich target kinds: `url`, `command`, `commandGroup`, `protected`.
- Structured report schema with selector, screenshot, console, and page error details.
- Mutation-to-contract mapping and applicability reporting.
- Realistic KubeStellar Console example.

## v0.3

- Provider adapter interfaces for Percy, Chromatic, Argos, and Applitools.
- Baseline review/update command.
- Rich Playwright trace extraction.
- Route/contract discovery helpers.

## v1.0

- Stable public config/report schemas.
- Trusted GitHub issue creation workflow.
- Production-grade provider adapters.
- Monorepo-scale planning and cost budgets.
